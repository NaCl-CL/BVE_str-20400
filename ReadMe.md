# 東武20400系ストラクチャ
東武20400系(204010型,204020型)の車両ストラクチャです。
<br>※元5ドア車は含んでいません。
<br>行先が不揃いなので適当に書き換えてください

このストラクチャはBVE-trainsim5/6用に制作されたものです。<br>
BVE5/6での使用に関しては本文書の利用条件の下でご自由にお使いいただけます。<br>
なお、本データの著作権は放棄しておりません。フリー素材ではございませんのでご注意ください。<br>
また、BVE以外での使用は固くお断りいたします。<br>

## 利用条件
1.トラブル防止のため、改造/無改造に問わずデータを公開する際にReadMeへ”本データを使用した旨、**明記**をお願い致します。<br>
2.**無改造での単体配布は認めません。**<br>
3.改造の上での単体配布に関しては、事前にご相談ください。<br>
4.改造・無改造問わず、路線組み込みに関しては連絡不要でご自由にお使いいただけます。<br>
5.**BVE-trainsim5/6以外の他ゲームへの転用・転載は固く禁じます。**<br>

※そのほか不明な点がありましたら、下の連絡先にご連絡ください。<br>
※また、必要であれば、mqozファイルの提供(現状渡し)も可能です。ご連絡ください。

## 編成について

編成例<br>
←浅草(進行方向)　東武日光→
### 21411f
24410,23420,22420,21410
### 21422f
24420,23420,22420,21420
<br>
※21411fの23410ト22410は23420と22420で代用して下さい
